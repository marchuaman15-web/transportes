# ECOSEM HUARAUCACA — Sistema de Gestión de Órdenes de Trabajo v6.0

Sistema completo de gestión de órdenes de trabajo para el departamento de mantenimiento de ECOSEM HUARAUCACA.

## 🚀 Características

- ✅ Sistema de autenticación con 52 usuarios (mecánicos, supervisores, admin, gerencia)
- ✅ Gestión completa de OTs con Firebase Firestore
- ✅ Firma digital para mecánicos
- ✅ Sistema de impresión de OT (formato oficial)
- ✅ Dashboard ejecutivo con KPIs
- ✅ Calendario de mantenimiento preventivo
- ✅ Control de inventario de repuestos
- ✅ Exportación a Excel y PDF
- ✅ Funciona offline (PWA)

## 🔧 Tecnologías

- Frontend: HTML5 + CSS3 + JavaScript vanilla
- Backend: Firebase Firestore
- PWA con Service Worker
- Responsive design

## 📱 Despliegue

## 👥 Roles y Permisos

- **Admin**: Control total del sistema
- **Supervisor**: Gestión de OTs y mecánicos
- **Gerencia**: Vista ejecutiva (solo lectura)
- **Mecánicos**: Registro de OTs y firma digital

## 🔐 Seguridad

- Autenticación local con hash de contraseñas
- Sistema de permisos por rol
- Firma digital con canvas HTML5
- Datos encriptados en Firebase
